~~~
x = torch.empty(4, 2)
print(x)
~~~
>tensor([[1.7878e+23, 9.2906e-43],
        [1.7878e+23, 9.2906e-43],

무작위로 초기화된 행렬(텐서)
~~~
x = torch.rand(4, 2)
print(x)
~~~
>tensor([[0.8871, 0.3499],
        [0.3245, 0.5496],
        [0.2350, 0.4584],
        [0.7282, 0.1444]])

dtype dl long, 0으로 채워진 텐서
~~~
x = torch.zeros(4, 2, dtype=torch.long)
print(x)
~~~
>tensor([[0, 0],
        [0, 0],
        [0, 0],
        [0, 0]])
